Amazon Affiliate Links

Como os links são gerados e como você ganha comissão

Como funciona

1. Usuário clica em "Comprar na Amazon Brasil"
2. Site redireciona para amazon.com.br/s?k={título+autor}&tag={seu-id-afiliado}
3. Se usuário comprar qualquer coisa nos próximos 24h, você recebe comissão
4. Comissão típica de livros na Amazon.com.br: 4–6%

Exemplo de link gerado

https://www.amazon.com.br/s?k=O+Código+Da+Vinci+Dan+Brown&tag=acheumlivro-20

Sem necessidade de API da Amazon — apenas URLs de busca com o tag de afiliado. Simples e sem aprovação extra.

O que você precisa fazer

Pré-lançamento
  1. Cadastrar no Amazon Associados BR
  2. Criar seu ID de afiliado (ex: acheumlivro-20)
  3. Adicionar o ID como variável de ambiente no Vercel
Requisito da Amazon

A Amazon exige que o site tenha 3 vendas qualificadas nos primeiros 180 dias para manter a conta ativa. Com bom SEO, isso é facilmente atingível.
