Data Flow

User searches → API Route queries Google Books → results returned with filters applied
User clicks book → Book Detail Page loads (ISR cached) with cover, description, similar books
User clicks "Comprar" → redirected to amazon.com.br search with affiliate tag
Google crawls → hits pre-built category + book pages with full meta tags & schema markup
