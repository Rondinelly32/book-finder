Pages & SEO Strategy

Every page designed to rank on Google.com.br

Site Structure

Static / ISR Pages (Google-crawlable)
  • / — Homepage
  • /categoria/suspense
  • /categoria/romance
  • /categoria/ciencia
  • /categoria/... (auto-generated)
  • /livro/[id]/[slug] — Book detail
  • /sitemap.xml — Auto-generated
  • /robots.txt
Dynamic (client-side, not indexed)
  • /busca?q=... — Search results
  • Filter state (genre, pages, origin)
SEO Per Page
  • Title tag in Portuguese
  • Meta description with keywords
  • og:image (book cover)
  • Schema.org Book markup
  • Canonical URLs

Target Keywords (examples)

📄 Category pages
"melhores livros de suspense", "livros de romance para ler"
📖 Book detail pages
"[título do livro] vale a pena", "livros parecidos com [título]"
🔍 Homepage
"encontrar livro para ler", "indicação de livro"
