Design Summary — Ache um Livro

Tudo que vamos construir

Stack
  • Next.js 14 (App Router)
  • TypeScript
  • Tailwind CSS
  • Vercel (hospedagem gratuita)
APIs & Dados
  • Google Books API (gratuita)
  • Amazon Associados BR (rondinellyc09-20)
  • Sem banco de dados — ISR como cache
Páginas
  • / — Homepage com busca
  • /busca — Resultados + filtros
  • /categoria/[genero] — ISR, indexável
  • /livro/[id]/[slug] — ISR, indexável
  • /sitemap.xml + /robots.txt
SEO
  • Título e descrição em português
  • Schema.org Book markup
  • og:image com capa do livro
  • Sitemap automático
  • URLs amigáveis com slug
Filtros de Busca
  • Gênero (suspense, romance, etc.)
  • Nº de páginas (curto / médio / longo)
  • Origem (nacional 🇧🇷 / internacional)
  • Disponível em português
Funcionalidades
  • Busca via Google Books API
  • Livros parecidos (Google Books recommendations)
  • Link afiliado amazon.com.br por livro
  • UI 100% em português
